ECOAE Extension - GUI Textures

The following GUI texture files need to be created (176x166 PNG format):

- estorage.png - EStorage Controller GUI background
- ecalculator.png - ECalculator Controller GUI background
- efabricator.png - EFabricator Controller GUI background

These should follow the standard Minecraft container GUI layout with:
- Dark gray background (0x808080)
- Player inventory area at the bottom
- Machine info area at the top
- Slot areas as needed

For reference, see vanilla Minecraft container textures or other GTNH mod GUIs.
